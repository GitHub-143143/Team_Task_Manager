Project: Team Task Manager

Description:
This is a full-stack web application where users can sign up, log in, create tasks, assign tasks, and track progress using a dashboard.

Features:
- User Authentication (Signup/Login)
- Task Creation and Assignment
- Dashboard showing total, pending, done, overdue tasks

Tech Stack:
- Frontend: React (Vite)
- Backend: Node.js, Express.js
- Database: MongoDB
- Authentication: JWT

Steps to Run:
1. Clone the repository
2. Run backend:
   cd team-task-manager
   npm install
   npm start
3. Run frontend:
   cd frontend
   npm install
   npm run dev

Deployment:
- Backend: Railway
- Frontend: Netlify
